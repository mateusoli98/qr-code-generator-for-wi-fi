# QR Code Generator for Wi-Fi

Access the project directory.

```bash
cd qr-code-generator-for-wi-fi
```

Install dependencies.

```bash
npm install
```

```bash
npm run dev
```

### Lint

```bash
npm run lint
```

### Typecheck

```bash
npm run typecheck
```

### Build

```bash
npm run build
```

### Test

```bash
npm run test
```
